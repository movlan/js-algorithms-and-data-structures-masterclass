# Performance of Objects and Arrays

## Objects

### Big O of Objects

Insertion - O(1)\
Removal - O(1)\
Search - O(n)\
Access - O(1)

### Big O of Object Methods

Object.keys - O(n)\
Object.values - O(n)\
Object.entries - O(n)\
Object.hasOwnProperty - O(1)

## Arrays

### Big O of Arrays

Insertion - It Depends\
Removal - It Depends\
Search - O(n)\
Access - O(1)

### Big O of Array Methods

- push - O(1)
- pop - O(1)
- shift - O(n)
- unshift - O(n)
- concat - O(n)
- slice - O(n)
- splice - O(n)
- sort - O(n log n)
